# vehicle-pcbs

PCB design files for vehicles and accessories used by AERPAW project.

Designs:

- `LAM-original` contains projects (carrier, PDB) for original LAM revisions (<2025)
- `LAM-simplified` contains updated PDB (power distribution board) (>2025)
- `LightBoard` contains design for auxilary LED drivers for SAM revisions
- `SAM-original` contains projects for original SAM revisions carrier (<2026)
- `SAM-simplifier` contains projects for updated SAM revision (PDB only)

Util:

- `kicad-libs` is a unified symbol/footprint library for all projects. Configure future projects to point at this!
